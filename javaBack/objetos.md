# Objetos

Objetos sao utilizados para armazenar grupos mais complexos de informações, ele reflete as coisas da vida real. Ele sempre é composto por conjuntos de chave e valor . Exemplo de como criar um objeto:

```javascript
const objPessoa = {
    nome: 'Jose Silva',
    idade: 32,
    cpf: '111111111111',
    turma: 'java'
}
```
