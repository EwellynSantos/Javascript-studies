# If e else



### Estrutura tradicional

```javascript
if (condition) {
  // código a ser executado se a condição for verdadeira
} else {
  // código a ser executado se a condição for falsa
}
```

### Operador ternário

```javascript
condition ? expressionIfTrue : expressionIfFalse;
```
